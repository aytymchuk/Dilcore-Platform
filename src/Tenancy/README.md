# Tenancy Domain

Enables multitenancy capabilities within the platform. This domain manages tenants and controls user access to specific tenant environments.

## Projects

*   **Tenancy.Domain**: Core DDD models (Aggregates, Entities, Value Objects).
*   **Tenancy.Core**: Business logic and use cases.
*   **Tenancy.Store**: Data persistence and repositories.
*   **Tenancy.Infrastructure**: External service integrations.
*   **Tenancy.WebApi**: Minimal API endpoints.
*   **Tenancy.Actors.Abstractions**: Orleans actor interfaces.
*   **Tenancy.Actors**: Orleans actor implementations.
